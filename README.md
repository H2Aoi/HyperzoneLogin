# HyperZoneLogin 个性化修改
自用
实现多种登入形式的框架

## 已支持的登入方式

- 类AuthME离线登入
- 外置登入
- 正版登入

## 兼容情况

### 版本兼容情况

低于1.19.1:  
域名前加o-或者offline识别为离线玩家  
高于1.19.1:  
直接进入即可

### 启动器兼容情况

- [x] HMCL
- [x] PCL
- [x] 原版启动器
- [x] Zalith

## 自动构建（GitHub Actions）

仓库已添加自动构建流程：每次推送到 `master` 会自动执行 `./gradlew build shadowJar`，并把构建产物提交到仓库的 `dist/` 目录（同时也会在 Actions 里上传 artifact）。
